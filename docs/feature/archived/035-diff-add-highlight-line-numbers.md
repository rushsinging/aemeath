## 需求

LLM 输出 unified diff 时，提升 diff 内容的可读性和定位能力：

1. 对 `+` 开头的 add 内容按目标文件语言做语法高亮。
2. 每行前缀显示 old/new 文件行号，方便定位变更位置。

## 实现结果

- unified diff 中 add 行已支持按目标文件语言做语法高亮。
- diff 每行前缀已显示 old/new 文件行号。
- 该能力用于提升 LLM 输出代码变更时的阅读体验和定位效率。
