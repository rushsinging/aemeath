# Feature #54：主动压缩触发：大上下文下防止 LiteLLM 代理拒绝

| 字段 | 值 |
|------|-----|
| 优先级 | 高 |
| 归档日期 | 2026-06-02 |

## 背景

长会话中 20+ 轮 tool calling 会累积大量 `tool_use` / `tool_result` 与消息，请求体过大时 LiteLLM 代理可能返回安全拒绝。需要在上下文过大时主动触发 compaction，降低请求体规模。

## 完成内容

1. 在上下文接近阈值时主动触发 compaction。
2. 降低大上下文请求体规模，减少代理端安全策略误拒绝风险。
3. 与 auto-compact 能力配合，长会话可继续推进。
