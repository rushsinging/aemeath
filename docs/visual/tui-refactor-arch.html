DDD 分层对接

Domain Layer — crates/runtime/
领域模型（Aggregate Root）· chat::domain · Domain Service · Repository Port
│ runtime::api::{ChatDto, ConfigDto, SessionDto}
▼
Application Layer — crates/runtime::api
公开契约：DTO、Error、Port 接口。TUI 不直接访问 Domain Model。
│ runtime::api::* 调用
▼
Adapter Layer — apps/cli/src/tui/
展示适配器：State（DTO 视图投影）+ Render（ratatui 渲染）+ Cmd Exec（副作用桥接）
