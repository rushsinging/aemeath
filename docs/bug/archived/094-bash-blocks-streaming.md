# Bug #94：Bash 工具运行时阻塞 LLM 流式渲染

| 字段 | 值 |
|------|-----|
| 优先级 | 中 |
| 发现日期 | 2026-05 |
| 归档日期 | 2026-05-30 |
| 状态 | 已确认修复 |
| 根因类别 | LLM 流式渲染 / Bash 工具执行阻塞 event loop |

## 症状

LLM 调用 Bash 工具执行长耗时命令期间，TUI 中 LLM 流式输出（assistant text streaming）暂停，直到 Bash 命令返回后才一次性继续渲染，体验上看起来 LLM 卡住。

## 根因

Bash 工具执行未与 LLM 事件流解耦，工具调用占用 event loop 后续 token 渲染受阻。

## 修复

在 G2 / 渲染管线重构系列中对工具执行与 LLM 事件流解耦，Bash 工具运行不再阻塞 streaming 渲染。
