# Feature #21: TUI 优化 Agent 调用输出展示

- **完成日期**：2026-05
- **归档日期**：2026-05-14
- **状态**：已确认完成

Agent 子任务每个 turn 仅显示工具名列表，噪声大。改为按工具+目标/参数摘要分组、合并连续同工具调用、按阶段分段，并提供折叠展开。

## 完成内容

- 按工具+目标/参数摘要分组显示
- 合并连续同工具调用
- 按阶段（探索/编辑/验证）分段
- 提供折叠展开功能
