# Bug #103：EnterWorktree/ExitWorktree 在 TUI 中显示原始 JSON 参数内容

| 字段 | 值 |
|------|-----|
| 优先级 | 中 |
| 发现日期 | 2026-05-31 |
| 归档日期 | 2026-06-02 |
| 状态 | 已确认修复 |
| 修复 | 4971564, 83f27e5 |

## 症状

在 TUI 中触发 `EnterWorktree` / `ExitWorktree` 工具调用时，工具调用块会显示原始 JSON 参数内容；`ExitWorktree` 无参数时也显示 `{}`，造成界面噪音。

## 根因

工具调用展示层对所有工具统一渲染 `input` / `arguments` JSON，`EnterWorktree` / `ExitWorktree` 缺少按工具的语义化摘要展示逻辑。

## 修复

为 `EnterWorktree` 和 `ExitWorktree` 注册专用 TUI 展示：`EnterWorktree` header 展示 branch/path 摘要，不再渲染原始 JSON detail；`ExitWorktree` 空参数不再显示 `{}`。

## 验证

- `cargo test -p cli worktree`
- 用户确认修复。
