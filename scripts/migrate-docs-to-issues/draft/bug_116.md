# Bug #116：TaskListCreate 工具返回未带 task list ID

| 字段 | 值 |
|------|-----|
| 优先级 | 中 |
| 发现日期 | 2026-06 |
| 归档日期 | 2026-06-04 |
| 状态 | 已确认修复 |
| 修复 | d2c9455c |

## 症状

`TaskListCreate` 工具调用成功后，返回结果中未包含新创建的 task list ID，导致 LLM 无法在后续 `TaskCreate` 调用中正确引用该 task list。

## 根因

`TaskListCreate` 工具返回格式不完整，未在结果中明确输出 task list ID。

## 修复

修复 `TaskListCreate` 返回格式，在结果中明确输出 task list ID，并增加引导说明，提示 LLM 在后续 `TaskCreate` 中使用该 ID。

## 验证

- 用户确认修复。

## 涉及路径

- `agent/features/tools/`（TaskListCreate 工具实现）
