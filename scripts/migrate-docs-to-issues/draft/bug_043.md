# Bug #43: TaskUpdate 使用全局 id 但 TUI task list 显示局部编号，agent 引用编号不一致

- **发现日期**：2026-05
- **归档日期**：2026-05-14
- **状态**：已确认修复
- **优先级**：高

## 症状

TUI task list 显示 batch 内局部编号（#1/#2/...），但 TaskUpdate 接受全局 task id，agent 看到的 tool 输出是全局 id，导致编号不一致。

## 修复

TaskStore 新增 display.rs 双向映射（resolve_display_id / format_display_id），TaskUpdate/TaskCreate/TaskList/TaskGet/TaskOutput 统一使用 batch 局部编号。

## 涉及路径

- `aemeath-core/src/task/display.rs`
- `aemeath-tools/src/task_update.rs`
- `aemeath-tools/src/task_create.rs`
- `aemeath-tools/src/task_list.rs`
- `aemeath-tools/src/task_get.rs`
- `aemeath-tools/src/task_output.rs`
