# Bug #55: 行内代码（`...`）自动换行处渲染异常

- **发现日期**：2026-05
- **归档日期**：2026-05-22
- **状态**：已确认修复
- **优先级**：中

## 症状

行内代码（反引号包裹）内容较长触发终端自动换行时，行内代码样式丢失或截断。

## 根因

渲染路径先用剥离 Markdown 标记后的纯文本计算 wrap，再对每个 wrap chunk 单独调用 `inline_markdown_spans`。当反引号标记或行内代码内容跨越 wrap 边界时，单个 chunk 内标记不完整，导致样式丢失/截断。

## 修复

Markdown 普通行在无选区渲染时先解析完整行的 inline spans，再按终端宽度切分 span，保留跨行样式；选区路径继续使用纯文本以保证复制/高亮坐标稳定。补充行内代码跨边界、标记卡边界、空字符串回归测试。
